# 📤 ربات صف‌کن و ارسال‌گر محتوا (KV Dumper)

رباتی Serverless بر پایه Cloudflare Workers برای ساخت صف (Queue) از عکس‌ها، ویدیوها و متن‌ها و ارسال آن‌ها به کانال تلگرام با تاخیر زمانی مشخص. این ربات برای زمان‌بندی پست‌های کانال بدون نیاز به سرور پیچیده بسیار کاربردی است.

---

## ✨ امکانات و ویژگی‌ها

- 📥 **ذخیره مستقیم در صف:** ارسال عکس، ویدیو یا متن به ربات تا در صف انتظار قرار گیرد.
- 🚀 **ارسال با تاخیر:** ارسال تمام محتواهای صف به کانال با تاخیر ۱۰ ثانیه‌ای بین هر پست (برای جلوگیری از محدودیت تلگرام).
- 🗑 **مدیریت صف:** پاکسازی کل صف با یک کلیک.
- 📊 **آمار دقیق:** نمایش تعداد محتواهای در انتظار ارسال.
- ⏱ **جلوگیری از Timeout:** استفاده از `ctx.waitUntil` برای پردازش صف‌های طولانی بدون قطع شدن توسط کلودفلر.
- 🛡 **دیتابیس D1:** ذخیره‌سازی امن و سریع محتواها به جای محدودیت‌های KV.

---

## 🛠 راهنمای نصب و راه‌اندازی

### مرحله ۱: پیش‌نیازها در کلودفلر
1. ساخت یک Worker در Cloudflare.
2. ساخت یک دیتابیس در Cloudflare D1.
3. اضافه کردن Variables های زیر در تنظیمات Worker:
   - `BOT_TOKEN` (توکن ربات)
   - `ADMIN_ID` (ایدی عددی ادمین)
   - `CHANNEL_ID` (ایدی عددی کانال)

### مرحله ۲: ساخت جداول دیتابیس
وارد کنسول D1 شوید و دستورات زیر را اجرا کنید:

```sql
CREATE TABLE IF NOT EXISTS queue (id INTEGER PRIMARY KEY AUTOINCREMENT, type TEXT, content TEXT, caption TEXT);
CREATE TABLE IF NOT EXISTS settings (key TEXT PRIMARY KEY, value TEXT);
INSERT OR IGNORE INTO settings (key, value) VALUES ('bot_status', 'on');
```

### مرحله ۳: راه‌اندازی
1. کد را روی ورکر دیپلوی کنید.
2. آدرس `https://your-worker-url/setwebhook` را در مرورگر باز کنید.
3. در ربات `/start` را بزنید.
4. چند عکس/ویدیو/متن برای ربات بفرستید تا در صف قرار گیرند.
5. روی دکمه "🚀 ارسال تمام محتواها به کانال" بزنید.

## 🎛 پنل مدیریت (ربات تلگرام)

| دکمه | عملکرد |
| :--- | :--- |
| 🚀 ارسال تمام محتواها | شروع پردازش صف و ارسال به کانال (با تاخیر ۱۰ ثانیه‌ای) |
| 🗑 پاکسازی صف | حذف تمام محتواهای در انتظار |
| 📊 آمار صف | نمایش تعداد محتواهای ذخیره شده |
| 🔌 روشن/خاموش | متوقف کردن ربات |
